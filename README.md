# simple-crud-py-api
This project is a simple CRUD API developed in Python

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install `simple-crud-py-api`.

```bash
pip install simple_crud_py_api
```

## Usage

```python
import simple_crud_py_api
```
